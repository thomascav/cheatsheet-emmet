HTML SHORTCUTS

x>y Permet de placer une balise y dans une balise x
x>y*n Permet de placer n balises y dans une balise x
x+y+z place dans l'ordre, à la ligne des balises x, y et z
x>y.item$*5 Permet de placer 5 y de class item numérotés de 1 à 5 dans une balise x. Le nombre de $ determine le nombre de chiffre de numérotation
#x / p#x place une id x dans une div / place une id x dans un balise p
.x / p.x Place une class x dans une div / place une class x dans un balise p
a{xxx} Ajoute le texte xxx entre deux balises a
link / link:css Ajoute une ligne link pour le css avec href vide / ajoute la même ligne avec le href renseigné
img / link:css Ajoute une balise img
link / link:css Ajoute une ligne link pour le css avec href vide / ajoute la même ligne avec le href renseigné
link / link:css Ajoute une ligne link pour le css avec href vide / ajoute la même ligne avec le href renseigné

CSS SHORTCUTS

pos / pos:s Ajoute l'attribut "position" / ajoute l'attribut avec la valeur static
fl /fl:n Ajoute l'attribut float (par défaut left) / ajoute l'attribut avec la valeur none
cl /cl:n Ajoute l'attribut clear (par défaut both) / ajoute l'attribut avec la valeur none
d /d:ib Ajoute l'attribut display (par défaut block) / ajoute l'attribut avec la valeur inline-block
m (mt mr mb ml) Ajoute l'attribut margin (top right bottom left)
m (pt pr pb pl) Ajoute l'attribut paddin (top right bottom left)
h / w (:a) Ajoute l'attribut height / width (valeur auto)
f / fs / fz / ff Ajoute l'attribut font / font-style / font-size / font-family
ta Ajoute l'attribut text-align (par défaut left)
bg Ajoute l'attribut background
bgc Ajoute l'attribut background-color
c (:r :ra) Ajoute l'attribut color (rgb rgba)
bd Ajoute l'attribut border
bdrs Ajoute l'attribut border-radius
lis / list Ajoute l'attribut list-style / list-style-type
